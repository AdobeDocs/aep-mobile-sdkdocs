# Brand Concierge

Adobe Brand Concierge guides customer journeys with personalized and conversational discovery.

For more information, see the [official documentation](https://business.adobe.com/products/brand-concierge.html).

**Required modules:** Mobile Core (AEPCore), Edge Network (AEPEdge), Edge Identity (AEPEdgeIdentity), Brand Concierge (AEPBrandConcierge).
