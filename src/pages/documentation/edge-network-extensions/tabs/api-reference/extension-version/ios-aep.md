#### Swift

**Syntax**

```swift
static let extensionVersion
```

**Example**

```swift
let extensionVersion = Edge.extensionVersion
```

#### Objective-C

**Syntax**

```objectivec
+ (nonnull NSString*) extensionVersion;
```

**Example**

```objectivec
NSString *extensionVersion = [AEPMobileEdge extensionVersion];
```
