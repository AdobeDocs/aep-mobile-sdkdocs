#### Objective-C

**Syntax**

```objc
(void) setLogLevel: (ACPMobileLogLevel) logLevel;
```

**Example**

```objc
#import "ACPCore.h"

[ACPCore setLogLevel: ACPMobileLogLevelVerbose];
```

#### Swift

**Syntax**

```swift
(void) setLogLevel: (ACPMobileLogLevel) logLevel;
```

**Example**

```swift
import ACPCore

ACPCore.setLogLevel(ACPMobileLogLevel.verbose);
```
