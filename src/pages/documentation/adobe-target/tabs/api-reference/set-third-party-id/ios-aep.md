#### Swift

**Syntax**

```swift
static func setThirdPartyId(_ id: String)
```

* _id_ : a `String` that contains the custom visitor ID to be set in Target.

**Example**

```swift
Target.setThirdPartyId("third-party-id")
```

#### Objective-C

**Example**

```objc
[AEPMobileTarget setThirdPartyId:@"third-party-id"]
```
