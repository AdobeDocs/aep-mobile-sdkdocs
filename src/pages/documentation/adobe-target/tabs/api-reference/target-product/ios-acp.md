#### ACPTargetProduct

This class contains productId and categoryId.

#### Objective-C

**Syntax**

```objc
@interface ACPTargetProduct : NSObject

/* Product ID */
@property(nonatomic, strong, nullable) NSString* productId;

/* Category ID */
@property(nonatomic, strong, nullable) NSString* categoryId;
@end
```
