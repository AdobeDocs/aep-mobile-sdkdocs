<InlineAlert variant="warning" slots="text"/>

Use caution when manually clearing the queue. This operation cannot be reverted.

#### Swift

**Syntax**

```swift
static func clearQueue()
```

**Example**

```swift
Analytics.clearQueue()
```

#### Objective-C

**Syntax**

```objc
+ (void) clearQueue;
```

**Example**

```objc
[AEPMobileAnalytics clearQueue];
```
