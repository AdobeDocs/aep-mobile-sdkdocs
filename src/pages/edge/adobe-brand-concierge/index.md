# Adobe Brand Concierge

## Overview

The Brand Concierge extension provides an in-app conversational UI (a chat surface) that can be embedded into a host app with minimal UI wiring. The UI is connected to the Adobe Experience Platform by using the SDK's shared state (Configuration and Edge Identity) to derive the service configuration needed to run a session and enable brand controlled experiences through configuration and theming.

For more information, see the [official documentation](https://business.adobe.com/products/brand-concierge.html).
